项目功能介绍：
一．查看所有留言界面
用户可以在项目首页查看所有用户的留言，并可点击登录或注册


二．注册界面
注册时，用户需要输入姓氏，名字，邮箱地址（以便找回密码），用户名，密码并再次确认密码，点击注册账户即可跳转到登录界面（若已用账户，点击立即登录）

三．登录界面
输入注册时用的用户名或邮箱和密码即可，点击登录即可进入自己的留言管理面板，若忘记密码点击忘记密码进入密码重置界面

四．密码重置界面
输入注册账号时用的邮箱点击发送验证码，收到验证码后输入验证码，点击验证进入重置密码界面，输入新密码并再次确认，点击确认修改。

5.发布留言界面
点击发布留言进入发布留言界面，填写名字和留言内容点击发布留言即可发布自己的留言。

6.删除留言界面
点击我的留言可以查看自己发布的所有留言并可以删除自己发布的留言，
